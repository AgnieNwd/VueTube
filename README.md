# How to use this API
## Docker
Install Docker 
```
run script regen.sh
```
## Usefull commands
```docker ps``` : Show docker process

```docker exec -it "nom_container" bash``` : Go into the selected container

```docker logs "nom_container"``` : Show container logs

```docker-compose up -d``` : run the docker-compose file
